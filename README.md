# 土壤光谱指标预测与深度学习分析

基于土壤高光谱数据，使用深度学习模型对多项土壤指标（pH、OC、N、P、K、CEC等）进行回归预测与可解释性分析。

## 项目内容
- 土壤光谱数据预处理（SG平滑、SNV标准化、导数变换）
- 基于ResNet18/34/50的多指标回归预测
- 特征重要性分析（SHAP）
- 数据降采样、1×1卷积消融实验
- 模型评估（R²、RMSE）

## 技术栈
Python、PyTorch、Pandas、NumPy、Matplotlib、SHAP

## 项目成果
- 构建高精度土壤属性预测模型
- 完成光谱特征可解释性分析
- 输出完整实验报告与可视化结果
